# 01. 게임 개요

## 게임명

**ALL-IN** (올인) — 부제: *Bloodbet*

## 한 줄 요약

HP를 걸고 공격 성공 확률을 올리는 3분짜리 1:1 카드 도박 게임.

## 디자인 목표

| 목표 | 측정 기준 |
|---|---|
| 단시간 플레이 | 한 판 평균 3분 이내 |
| 도파민 | 리스크-리워드 베팅, 크리티컬 터짐, 역전 가능성 |
| 낮은 진입장벽 | 마나 시스템 없음, 자원 일원화(HP) |
| 반복 플레이성 | 매 판 다른 빌드(부운), 카운터픽 메타 |
</content>
## 핵심 룰

### 리소스

- **HP**: 시작 100. 0 이하가 되면 패배
- **마나 시스템 없음**: HP가 유일한 자원
- **베팅**: 카드 사용 시 0~10 HP를 추가로 지불해서 효과 증폭

### 베팅 메커니즘

카드마다 베팅 효과가 다르게 적용:

| 카드 타입 | 베팅 효과 |
|---|---|
| **Hit (명중형)** | 베팅 1당 명중률 +X% (빗나가면 0뎀) |
| **Crit (치명타형)** | 베팅 1당 크리티컬률 +X% (항상 맞고, 크리 시 배수) |
| **Fixed (확정형)** | 베팅 1당 데미지 +1 (항상 맞음, 안정적) |
| **Utility (유틸)** | 대부분 베팅 불가 |

베팅은 **즉시 자기 HP에서 차감**. 빗나가도 HP는 돌아오지 않음.

### 턴 구조

1. **턴 시작**: 핸드 5장 보충 (덱에서 드로우)
2. **카드 사용**: 최대 2장 사용 가능 (부운으로 3장까지 가능)
3. **베팅 결정**: 각 카드마다 0~10 HP 베팅
4. **효과 발동**: 명중/크리/확정 판정 후 데미지/효과 적용
5. **턴 종료**: 상대 턴으로

### 덱 구조

- 각 직업당 15장 고정 덱
- 공격 8장 / 유틸·방어 5장 / 시그니처 2장
- 시그니처는 **1게임 1회**만 사용 가능
- 시그니처는 **게임 시작 3턴 이후부터** 사용 가능 (첫 턴 올인 방지)
- 같은 턴에 **시그니처 1장만** 사용 가능

### 승리 조건

- 상대 HP 0 이하
- 15턴 시간초과 시 HP 높은 쪽 승리 (동률은 무승부)
</content>
## 픽 단계 (게임 시작 전)

1. **선픽/후픽 결정**: 랜덤 (주사위 또는 가위바위보)
2. **선픽 직업 선택**: 3개 직업 중 1개
3. **후픽 직업 선택**: 남은 2개 중 1개 (미러전 금지)
4. **선픽 부운 선택**: 10종 부운 풀에서 랜덤 제시된 3개 중 1개
5. **후픽 부운 선택**: 3개 중 1개 (도박사 직업 픽 시 4개 중 1개, 패시브)

후픽 어드밴티지: 상대 직업을 알고 직업/부운을 카운터 픽할 수 있음.

## 도파민 장치

- **베팅 콜 직전의 긴장감**: 풀베팅하면 터질까 빗나갈까
- **크리티컬 연출**: 2.5~3.5배 데미지 순간 폭발
- **역전 메커니즘**: HP 30 이하에서만 사용 가능한 강력 카드 (처형, 최후의 발악)
- **시그니처 컷인**: 1게임 1회 한정의 특수 연출
- **부운 다양성**: 매 판 다른 빌드 → 새로운 전략 발견

## 설계 결정 배경

### 왜 HP 베팅인가? (마나 시스템 포기)

- **자원 일원화**: 규칙 절반으로 축소, 진입 장벽 ↓
- **모든 결정의 무게**: "마나 손해" vs "내 목숨 깎기" — 심리적 무게 차원이 다름
- **자연스러운 페이싱**: HP 낮을수록 베팅 여력 ↓ → 게임 자동 종료

### 왜 미러전 금지인가?

- 같은 직업 거울전은 전략 다양성 없음 (시뮬 결과 평균 턴 수 급증)
- 3직업 상성 순환 구조가 게임의 재미 포인트 — 미러는 이걸 무효화
- 선/후픽 시스템으로 자연스럽게 메타 게임 발생

### 왜 시그니처 3턴 제한인가?

1차 시뮬에서 도박사가 첫 턴에 시그니처 2장 올인 콤보로 115뎀을 뽑아 게임이 즉사함. 페이싱 확보 + 카드 게임의 빌드업 감성 유지.
