# 04. 부운 (Boon) 10종

## 시스템 개요

매 게임 시작 시 10종 부운 풀에서 랜덤으로 **3개** 제시 → **1개** 선택해서 그 판에 적용.
도박사 직업 선택 시 패시브로 **리롤 1회** 가능 (제시된 3개를 전부 버리고 3개 다시 뽑기, 리롤 풀은 초기 제시와 독립).

### 설계 원칙

1. **모든 부운은 직업 무관하게 비슷한 가치** — 특정 부운 독점 방지
2. **부운마다 추구하는 빌드가 다름** — 다양성
3. **상대 직업에 따라 우선순위가 달라짐** — 메타 게임 형성

### 카테고리 분류

- 🟢 **안정형 (Steady)**: 생존력 ↑, 장기전에 유리
- 🔴 **공격형 (Aggressive)**: 딜 증폭, 단기 결착
- 🎲 **도박형 (Risky)**: 변수 창출, 하이리스크 하이리턴
- 🛠️ **유틸형 (Utility)**: 구조적 이점, 콤보 빌드업

## 🟢 안정형 (Steady)

### BN01. 🩸 강철 심장
- **효과**: 시작 HP 100 → 130
- **카테고리**: 안정형
- **추천 매치**: 광전사 상대 (한 방 컷 방지), 컨트롤러 픽 시 시너지
- **시너지**: 인내의 반격(받은 데미지 ↑), 피의 대가(흡혈 절대량 ↑)

### BN02. 🛡️ 견고한 의지
- **효과**: 받는 모든 데미지 -2 (최소 1)
- **카테고리**: 안정형
- **추천 매치**: 도박사 상대 (크리 데미지 감쇠), 연쇄 베기 카운터
- **주의**: 크리 한 방 고데미지에는 효과 적음 (비율 감쇠 아님)

### BN03. 💚 회복의 가호
- **효과**: 매 턴 시작 시 HP +5
- **카테고리**: 안정형
- **추천 매치**: 컨트롤러 픽 시 극강 시너지, 도박사 상대
- **주의**: 게임 길어질수록 강함, 광전사 단기 결착엔 약함

---

## 🔴 공격형 (Aggressive)

### BN04. ⚔️ 칼날의 축복
- **효과**: 모든 카드 데미지 +3 (명중/크리/확정 상관없이)
- **카테고리**: 공격형
- **추천 매치**: 광전사 (연쇄 베기 +9뎀, 분쇄 일격 +3뎀)
- **시너지**: 다회타 카드와 특히 강력 (연쇄 베기 30 → 39뎀)

### BN05. 🎯 정밀의 눈
- **효과**: 모든 명중률 +15%, 모든 크리율 +10%
- **카테고리**: 공격형
- **추천 매치**: 광전사·도박사 양쪽 유용
- **주의**: 컨트롤러 Fixed 카드에는 효과 없음

### BN06. 🔥 광기의 인장
- **효과**: 베팅 1당 추가 명중률/크리율 +1% (카드 자체 효과에 더함)
- **카테고리**: 공격형
- **추천 매치**: 광전사 (베팅 전략 극대화)
- **예시**: 분쇄 일격(+6%/베팅) + 인장(+1%) = +7%/베팅. 풀베팅 시 +10% 추가 명중

## 🎲 도박형 (Risky)

### BN07. 🎰 행운의 동전
- **효과**: 카드 사용 시마다 30% 확률로 데미지/효과 ×2
- **카테고리**: 도박형
- **추천 매치**: 도박사 픽 시 컨셉 시너지
- **주의**: "반반" 확률 게임이 더 강해짐. 터지면 120뎀 이상도 가능

### BN08. 💸 피의 환원
- **효과**: 베팅한 HP의 50%를 즉시 회복 (베팅 효율 2배)
- **카테고리**: 도박형
- **추천 매치**: 광전사·도박사 (베팅 적극형)
- **주의**: 컨트롤러 픽 시 거의 무용지물 (베팅 안 함)

---

## 🛠️ 유틸형 (Utility)

### BN09. 🃏 풍요의 손
- **효과**: 매 턴 카드 사용 횟수 +1 (2 → 3장)
- **카테고리**: 유틸형
- **추천 매치**: 모든 직업
- **주의**: 게임 템포가 급격히 빨라짐. 시그니처 3턴 제한과 맞물려 4턴 게임이 가능

### BN10. 👁️ 예지의 눈동자
- **효과**: 시작 시 손패 +3 (일반 5장 → 8장), 핸드 사이즈 5 → 7
- **카테고리**: 유틸형
- **추천 매치**: 도박사 (더블다운 콤보 찾기 쉬워짐), 광전사 (함성+시그니처 콤보)
- **주의**: 유틸/방어 카드 비중 높은 컨트롤러는 상대적으로 이득 적음

## 검증 필요 사항

시뮬레이션으로 확인해야 할 것:

- [ ] 각 부운 픽률 ±20% 이내 (특정 부운 독점 방지)
- [ ] 부운별 승률 영향 50% ±10%
- [ ] 도박사 4개 중 1개 선택 시 승률 보정값 +3~7% 범위
- [ ] 동일 매치업에서도 부운 조합에 따라 결과 달라지는지

## 밸런싱 예상 문제

- **BN09 풍요의 손**: 게임 템포 너무 빠름 우려. 필요 시 "턴당 카드 3장, 단 시그니처 제외" 같은 제한 추가
- **BN01 강철 심장**: +30 HP가 크므로, 대신 모든 회복 효과 -20% 같은 페널티 고려
- **BN10 예지의 눈동자**: 도박사 콤보 빌드 너무 쉬워짐 우려. 핸드 사이즈만 +2로 조정 가능

1차 시뮬레이션에서 문제 발견 시 `docs/06-balance-notes.md`에 수정 히스토리 기록.
