# 02. 직업 3종

## 개요

3직업 상성 순환 구조 (가위바위보):

```
🔥 광전사 → (먹잇감) 🎲 도박사 → (먹잇감) 🛡️ 컨트롤러 → (먹잇감) 🔥 광전사
```

- 광전사: 단순 화력으로 도박사의 변수를 압도
- 도박사: 한 방의 운빨로 컨트롤러의 운영을 무너뜨림
- 컨트롤러: 안정적 방어와 회복으로 광전사의 자해를 유도

## 🔥 광전사 (Berserker)

### 정체성

> "내 피 1방울이 더 강하다."

HP 베팅으로 명중률을 미친 듯이 끌어올리는 극단적 공격수. 풀베팅하면 거의 확정타, 빗나가면 자기 손해.

### 패시브

- **광기의 각인**: HP 베팅 1당 추가 명중률 +2% (카드 자체 베팅 효과에 더함)
- 예: 분쇄 일격(베팅당 +6%) → 광전사 사용 시 베팅당 +8%

### 전투 스타일

- 공격 카드 비중 압도적 (8/15장)
- 대부분 Hit 타입 (명중률 게임)
- "함성 → 시그니처 → 처형" 같은 3장 콤보 빌드업
- 방어 카드는 `강철 피부` 1장뿐 — 맞으면서 때리는 스타일
- HP 낮을수록 강해지는 카드 多 (처형, 최후의 발악)

### 천적/먹잇감

- **천적**: 🛡️ 컨트롤러 — 확정타로 맞은 뒤 회복, 광전사의 베팅 효율 무력화
- **먹잇감**: 🎲 도박사 — 크리 운에 의존하는 도박사를 풀베팅 확정타로 압살

### 플레이감

"이번엔 진짜 죽일 수 있어" 하다가 자기가 먼저 죽음. 호쾌하고 단순.

## 🎲 도박사 (Gambler)

### 정체성

> "이번 판도 운이 좋네."

크리티컬과 부운 빌드 다양성이 무기. 카드 자체가 변수 많고, 매 판 다른 빌드로 상대를 당황시킴.

### 패시브

- **운명의 총아**: 부운 픽 단계에서 **리롤 1회**. 제시된 3개 부운이 마음에 안 들면 전부 다시 랜덤으로 뽑을 수 있음
- 빌드 다양성 우위. 상대가 광전사/컨트롤러일 때 카운터 부운 뽑힐 확률 ↑

### 전투 스타일

- 공격 카드 6장이 Crit 타입 (치명타 게임)
- 대부분 기본 데미지 낮지만 크리 시 2~3.5배
- 변수 카드 多: 잭팟(1d10), 야바위(랜덤 데미지), 올인(안 터지면 자해)
- 정보전 카드: 미래 보기, 손패 바꾸기 — 콤보 빌드업
- 독/회피/블러프 등 상태이상 카드

### 천적/먹잇감

- **천적**: 🔥 광전사 — 광전사의 확정 풀베팅을 크리 확률만으로는 못 이김
- **먹잇감**: 🛡️ 컨트롤러 — 컨트롤러의 점진적 회복을 한 방에 무너뜨림

### 플레이감

"제발 터져라..." 기도하며 버튼 누름. 터지면 환호, 안 터지면 자폭. 양극단.

### 설계 노트

패시브 변천:
- v1: "크리 데미지 ×3" → 시그니처 G15 와 컨셉 중복으로 폐기
- v2: "부운 선택지 3 → 4" → 선택지가 많아져도 대부분 명확히 한 장이 우세하면 의미 희석
- v3 (현재): **부운 리롤 1회** — 초기 3개 제시가 전부 취향 안 맞을 때만 쓰는 도박적 선택. "이 정도면 나쁘지 않은데 한 번 더?" 의 긴장감 추가. 초기 제시와 리롤 후 풀이 완전히 독립이라 리롤이 더 나빠질 수도 있음 (빌드의 도박성 강화)

## 🛡️ 컨트롤러 (Warden)

### 정체성

> "안 죽는 게 이김."

확정타 + 회복 + 디버프로 야금야금 상대를 말려 죽이는 운영형. 베팅은 거의 안 함.

### 패시브

- **감시의 눈**: 매 턴 시작 시 상대 손패 1장 무작위로 봄 (정보 우위)

### 전투 스타일

- 공격 카드 대부분 Fixed 타입 (확정 데미지)
- 데미지는 낮지만 안정적 (베팅 의존도 낮음)
- 방어/회복 카드 다수: 방패 벽 18뎀 차단, 신성한 보호 20HP 회복
- 디버프 카드: 결박, 약점 포착, 마나 봉인(베팅 상한 제한)
- 시그니처 2장 모두 상대 베팅/공격 카운터 전용

### 천적/먹잇감

- **천적**: 🎲 도박사 — 크리 한 방이 운영 다 무너뜨림
- **먹잇감**: 🔥 광전사 — 광전사의 자해 + 방어 + 인내의 반격으로 역전

### 플레이감

"천천히 말려 죽인다." 상대가 빡치게 만드는 재미. 하지만 거울전은 지루함 주의.

### 설계 노트

컨트롤러는 구조적으로 매치업 편차가 큼 (광전사 65% vs 도박사 20%). 
부운 시스템이 이 문제를 부분적으로 해결:
- 도박사 상대 시 `견고한 의지`(데미지 -2) 같은 부운으로 크리 견제
- 광전사 상대 시 베팅 무시 카드로 더 쉽게 운영

## 직업별 매치업 의도 (시뮬 목표치)

| 매치업 | 의도 승률 |
|---|---|
| 🔥 광전사 vs 🎲 도박사 | 광전사 60% |
| 🎲 도박사 vs 🛡️ 컨트롤러 | 도박사 60% |
| 🛡️ 컨트롤러 vs 🔥 광전사 | 컨트롤러 60% |

**현재 상태 (시뮬 2차, 부운 미적용):**
- 광전사 vs 도박사: 광전사 26% (의도 미달, 도박사 OP 상태)
- 광전사 vs 컨트롤러: 광전사 73% (의도 초과, 컨트롤러 약함)
- 도박사 vs 컨트롤러: 도박사 96% (도박사 극OP)
