# 05. 게임 플로우

## 전체 진행 순서

```
[진입] → [방 생성/참가] → [대기실] → [게임 시작]
   ↓
[코인토스: 선/후픽 결정]
   ↓
[선픽: 직업 선택 (30초)] → [후픽: 직업 선택 (30초)]
   ↓
[게임판 전환]
   ↓
[부운 선택 (15초, 양쪽 동시)]
   ↓
[게임 진행: 턴제 배틀]
   ↓
[승패 결정] → [리플레이 or 재대전]
```

---

## 1. 사이트 진입

### 초기 화면

- **방 생성** 버튼
- **방 코드 입력** (참가자용)
- 닉네임 입력 (선택)

### 방 생성 플로우 (방장)

1. "방 만들기" 클릭
2. 고유 방 코드 생성 (6자리 영숫자, 예: `AB3F9K`)
3. 대기실 입장 → URL 공유 가능 (`/room/AB3F9K`)

### 방 참가 플로우 (참가자)

1. 방 코드 입력 또는 공유 URL 접속
2. 대기실 입장

---

## 2. 대기실

### 화면 구성

- 상단: 방 코드 표시 (클립보드 복사 버튼)
- 중앙: 플레이어 슬롯 2개 (방장 / 참가자)
- 하단: **게임 시작 버튼** (방장만 활성화, 2명 모두 입장 시 활성화)

### 기능

- 참가자 입장 시 방장에게 시각/청각 피드백
- 참가자가 나가면 슬롯이 다시 비워짐
- 닉네임 표시 또는 "Player 1/2" 기본값
</content>
---

## 3. 코인토스 (선/후픽 결정)

### 연출

- 화면 중앙에 코인 3D 애니메이션 (혹은 2D 스프라이트)
- 양쪽 플레이어 아바타가 마주보는 형태
- 코인이 회전하며 낙하 → 정지 → 앞/뒷면 공개
- "선픽" 텍스트가 결정된 플레이어 쪽으로 이동

### 기술 요소

- 결과는 **서버에서 결정** (클라이언트 조작 불가)
- 애니메이션은 약 3초
- 스킵 불가 (긴장감 유지)

## 4. 직업 선택 (선픽 → 후픽)

### 화면 레이아웃

```
┌─────────────────────────────────────────────────┐
│  ⏱ 30초 타이머                         [선픽 차례] │
├─────────────────────────────────────────────────┤
│                                                  │
│  ┌────────┐  ┌────────┐  ┌────────┐             │
│  │   🔥   │  │   🎲   │  │   🛡️   │             │
│  │ 광전사  │  │ 도박사  │  │ 컨트롤러 │             │
│  │ [1]    │  │ [2]    │  │ [3]    │             │
│  └────────┘  └────────┘  └────────┘             │
│                                                  │
│  [클릭 시 상세 정보 팝업]                          │
│                                                  │
└─────────────────────────────────────────────────┘
```

### 직업 상세 팝업 (카드 클릭 시)

```
┌──────────────────────────────────────────────────┐
│           🔥 광전사 (Berserker)          [X] 닫기 │
├──────────────────────────────────────────────────┤
│                   │ [간단 설명]                   │
│                   │ "내 피 1방울이 더 강하다."     │
│                   │ HP 베팅으로 명중률을 극대화   │
│   [직업 이미지]    │ 하는 극단적 공격수.          │
│                   │                              │
│                   │ [패시브: 광기의 각인]         │
│                   │ HP 베팅 1당 명중률 추가 +2%   │
│                   ├──────────────────────────────┤
│                   │ [덱 리스트 (15장)]            │
│                   │ ▸ B1 분쇄 일격 (Hit, 18뎀)    │
│                   │ ▸ B2 광폭한 돌진 (Hit, 25뎀)  │
│                   │ ▸ ... (스크롤 가능)           │
│                   │ ▸ ⚡ B14 피의 광기 (시그니처)  │
│                   ├──────────────────────────────┤
│                   │  [이 직업으로 시작하기]        │
└──────────────────────────────────────────────────┘
```

### 제한 시간 및 규칙

- **30초 내 선택** 필수
- 시간 초과 시 랜덤 직업 자동 선택
- 후픽은 선픽이 고른 직업이 **비활성화**(회색 처리 + 🔒 아이콘)
- 상호 미러전 방지 하드 룰

### 기술 요소

- 타이머는 서버 기준 (클라이언트 조작 불가)
- 선택 확정 후 서버에 전송 → 후픽에게 결과 브로드캐스트

## 5. 게임판 전환 및 부운 선택

### 화면 전환 연출

- 직업 확정 직후 화면이 페이드 아웃
- "전투 준비 중..." 텍스트와 로딩
- 게임판 화면으로 페이드 인 (약 2초)

### 게임판 레이아웃 (TCG 표준)

```
┌────────────────────────────────────────────────────┐
│         상대 (12시 방향)                            │
│  [상대 아바타] HP 100/100  [상대 부운 슬롯]         │
│                                                     │
│  [상대 덱] [상대 손패 뒷면 5장]  [상대 묘지]        │
│                                                     │
│  ─────────────── [필드 중앙] ───────────────       │
│                                                     │
│  [내 묘지]  [내 손패 앞면 5장]  [내 덱]            │
│                                                     │
│  [내 아바타] HP 100/100  [내 부운 슬롯]            │
│         나 (6시 방향)                               │
└────────────────────────────────────────────────────┘
```

### 부운 선택 페이즈

- 게임판 중앙에 **부운 선택 카드 3장** 팝업 (도박사는 4장)
- **양쪽 플레이어 동시에 독립 시행** (각자 다른 부운 3장)
- 상대가 뭘 고르는지는 선택 완료 전까지 비공개
- **15초 타이머** (공통)
- 시간 초과 시 랜덤 선택

### 부운 카드 UI

```
┌──────────────┐  ┌──────────────┐  ┌──────────────┐
│     🩸       │  │     ⚔️       │  │     🎰       │
│  강철 심장    │  │ 칼날의 축복   │  │  행운의 동전  │
│              │  │              │  │              │
│  HP 100→130  │  │  모든 카드    │  │  30% 확률    │
│              │  │  데미지 +3   │  │  효과 ×2     │
│              │  │              │  │              │
│   [선택]     │  │   [선택]     │  │   [선택]     │
└──────────────┘  └──────────────┘  └──────────────┘
```

### 선택 완료 후

- 양쪽 부운이 아바타 옆 부운 슬롯에 배치됨
- **Hover 시 상대 부운 툴팁 표시** (효과 공개)
- 패시브도 같은 방식으로 Hover 시 확인 가능
- 약 2초 대기 후 자동으로 첫 턴 시작

## 6. 턴 진행

### 턴 시작

1. 현재 플레이어의 HP에 매 턴 효과 적용 (회복의 가호 +5, 독 -4 등)
2. 덱에서 손패가 5장이 되도록 드로우
3. "내 턴" 인디케이터 표시
4. 카드 사용 가능 표시 (빛나는 테두리)

### 카드 사용 플로우

1. 손패에서 카드 **드래그 or 클릭**
2. 카드 상세 미리보기 (확대 뷰)
3. **베팅 슬라이더** 표시 (0~10 또는 카드별 상한)
   - 슬라이더 조작 시 실시간 명중률/크리율 미리보기
   - 예: "베팅 5 → 명중률 70%"
4. **확정 버튼** 클릭 → 애니메이션 발동
5. 결과 표시 (명중/빗나감/크리)

### 공격 연출

- **명중**: 공격 이펙트 + 데미지 숫자 플로팅
- **빗나감**: "MISS" 텍스트 + 연기 이펙트
- **크리**: 화면 플래시 + 큰 데미지 숫자 + 배경 진동
- **시그니처**: 풀스크린 컷인 + 캐릭터 대사 (예: "내 피로 쓴다!")

### 턴 종료

- 2장 사용 or "턴 종료" 버튼 클릭
- 상대 턴으로 넘어가는 화면 전환 (0.5초)

## 7. 게임 종료

### 종료 조건

- 한쪽 HP 0 이하 → 즉시 종료
- 15턴 시간 초과 → HP 높은 쪽 승리 (동률 무승부)

### 종료 연출

- **승자**: 화면 밝아짐 + "VICTORY" 텍스트 + 캐릭터 승리 포즈
- **패자**: 화면 어두워짐 + "DEFEAT" 텍스트 + 캐릭터 쓰러짐
- BGM 전환

### 결과 화면

```
┌──────────────────────────────────────┐
│            VICTORY                   │
├──────────────────────────────────────┤
│  턴 수: 5턴                          │
│  최대 베팅: 10 HP                    │
│  총 데미지: 115                       │
│  역전 횟수: 1회                      │
│  크리티컬 발동: 2회                   │
├──────────────────────────────────────┤
│  [재대전]  [방 나가기]                │
└──────────────────────────────────────┘
```

### 재대전 옵션

- **재대전**: 방 유지, 직업/부운 선택부터 다시
- **방 나가기**: 홈 화면으로

## 상태 관리 (서버 권위형)

### 이유

베팅 게임 특성상 **클라이언트 신뢰 시 치팅 즉시 발생**. 따라서 모든 게임 로직은 서버에서 처리.

### 서버 보관 상태

- 현재 턴, 현재 플레이어
- 양쪽 HP, 덱, 손패, 묘지
- 부운 상태, 시그니처 사용 여부
- 디버프/버프 상태 (독, 침묵, 베팅 상한 등)

### 클라이언트 → 서버 액션

- `room:create`, `room:join`
- `game:start`, `class:pick`, `boon:pick`
- `card:play` (card_id, bet_amount)
- `turn:end`

### 서버 → 클라이언트 이벤트

- `state:update` (전체 또는 차분)
- `card:result` (명중/빗나감/크리 + 애니메이션 트리거)
- `turn:changed`, `game:ended`

### 부정 방지

- 상대 손패는 **뒷면만 전송** (ID 노출 X)
- 예지/카드 카운팅 등 정보 카드는 서버에서 해당 플레이어에게만 전송
- 타이머는 서버 기준, 클라이언트 조작 시 경고 후 강제 진행
